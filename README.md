# Diplom-Game
Игра для дипломной работы, первый проект

Обучающая 3D-игра на Unity по основам программирования. Игровое пространство делится на три комнаты, каждая комната - отдельный уровень. На уровне располагаются интерактивные объекты, которые выдают задания. При получении задания открывается пользовательский интерфейс с текстом задания, полем ввода ответа и кнопкой для отправки ответа. Результат проверки отображается под кнопкой, после её нажатия. При правильных ответах на все задания (прогресс отслеживается в счётчике в верхнем левом углу) открывается доступ к следующему уровню. Первый уровень направлен на понимание логических операций, второй - на поиск и исправление ошибок в строке кода, третий - на ввод команд в псевдоинтерпретатор для настройки работы светофора.
